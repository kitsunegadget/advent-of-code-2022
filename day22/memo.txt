それぞれの面の空間方向、展開図と空間座標軸との関係をMapで管理
"0,0,-1" -> { netPos: "0,2", axis: ["+y", "+x"] }
...

それぞれの面の上下左右の接続をMapで表す
"0,2" -> { right: "2,3", down: "1,2", left: "1,1" up: "1,0" }
...

スタート面を　z = -1 とすると

                                               ( 0,  0, -1): [+y, +x]
 (0, -1, 0): [+z, -x] | (-1, 0, 0): [+z, +y] | ( 0,  1,  0): [+z, +x]
                                               ( 0,  0,  1): [-y, +x] | ( 1, 0, 0): [-y, -z]
